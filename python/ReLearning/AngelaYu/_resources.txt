PyCharm Features:
  - Spell Check: hover over squiggly likes to correct it from the suggestions 
  - More space: see multiple codes --> split screen --> right click on file name and select split and move right for side by side
  - Built-in Linter: hover over the squiggly lines to see suggested changes as per pep 8 styleGuide --> https://peps.python.org/pep-0008/
  - Local History: Last 12 Hours history  -->>  ToolBar --> VCS --> Local History --> Show History
  - View Structure: Structure Pane -->> breaks code into variables and function for easy access
  - Refactor Rename: Rename variable or function  all at once -->> RightClick on it and select Refactor --> Rename (make required changes) --> Do Refactor
  - TODO: tracking of todo tasks

 PyCharm Tips:
  To switch between opened files and tool windows, use the Switcher Ctrl+Tab.
  For navigation, press and hold Ctrl and use the Up and Down arrow keys or Tab and Shift+Tab, and Alt.
  Press Delete or BackSpace to close an editor tab or hide a tool window
